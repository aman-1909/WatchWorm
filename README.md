# Problem 

- Illegal  sand mining is a big problem in states like bihar, up, etc.
- It's depletes groundwater , increase flood vulnerability by lowering riverbeds.
- Destroy biodiversity.
- Fuels crimes and unregulation.

# Current Scenario 

- No effective way to detect such illegal mining in time which led to delay in law enforcement.

# My Proposed Solution 

- A  monitoring system which detect illegal sand mining near rivers using satellite / terrain data and AI.

# Impact 

- Protect rivers.
- Helps law enforcement agencies in monitoring.
- Reduce illegal mining and revenue loss.
- Improve environmental sustainability.
